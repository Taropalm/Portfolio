# Taro Website

Personal portfolio / profile page cho Taro.

## Cấu trúc
- `index.html` — nội dung chính (one-page)
- `css/style.css` — theme, layout, màu sắc chính
- `css/animations.css` — hiệu ứng, transition, keyframes
- `css/responsive.css` — media queries cho mobile/tablet
- `js/script.js` — tương tác (scroll, nav, animation trigger)
- `assets/images` — avatar, hình ảnh
- `assets/icons` — icon

## Roadmap
- v0.1 — HTML structure + CSS theme cơ bản + responsive + chạy local
- v0.2 — animation, glass card, hover, background effect
- v0.3 — nội dung thật (avatar, project, rank, social link)
- v1.0 — deploy GitHub Pages / Vercel
